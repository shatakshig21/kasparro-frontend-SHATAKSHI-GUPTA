# Kasparro – Frontend Engineering Assignment

## Overview
Kasparro is an AI-native SEO and Brand Intelligence platform designed for the AI-first search era (ChatGPT, Gemini, Perplexity).

This project demonstrates:
- Frontend system design
- Component-driven architecture
- Data modeling
- UX for data-heavy AI products

---

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Zustand
- shadcn/ui

---

## Folder Structure
app/ → Routes
components/ → UI & feature components
data/ → Mocked data
store/ → Zustand state
types/ → TypeScript models


---

## Key Features
- Public product website
- Authenticated dashboard (mocked)
- AI audit modules
- System architecture view
- Fully data-driven UI

---

## Architectural Decisions
- Modular component structure
- Centralized state via Zustand
- No hardcoded UI values
- Scalable data models
- Clear separation of concerns

---

## Tradeoffs
- No backend (mocked data only)
- No authentication
- Minimal animations for clarity
